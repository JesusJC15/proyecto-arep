# Frontend MVP

Base de interfaz para la segunda entrega del proyecto AREP. Presenta dos vistas principales:

- Portal del paciente para registrar sintomas y revisar la recomendacion.
- Panel profesional para revisar casos escalados con evidencia y trazabilidad.

## Scripts esperados

```bash
npm install
npm run dev
```

## Nota

La interfaz usa datos de demostracion y una organizacion lista para integrarse con el backend FastAPI definido en `../backend/`.
