# Base de conocimiento inicial

Este directorio contiene un corpus clinico minimo para respaldar el diseño del pipeline RAG del prototipo.

## Importante

- El contenido es de uso academico para prototipado.
- No representa una guia clinica validada para uso asistencial real.
- La tercera entrega debe reemplazar o fortalecer este corpus con fuentes verificadas y un proceso formal de curacion.
